# 📦 Indent Management Slip System

A web-based Indent Management System developed using **PHP and MySQL** that allows employees to request items digitally and enables administrators to manage and approve requests efficiently.

---

## 🚀 Features

* 🔐 Secure Login & Registration System
* 📝 Create Indent Request (Item, Quantity, Supplier, Purpose)
* 📊 Track Indent Status (Pending / Accepted / Rejected)
* 📄 View Indent Reports
* 👨‍💼 Admin Panel for Managing Requests
* 🔎 Filter Indents by Status
* ✏️ Update & Delete Indent Requests

---

## 🧠 System Workflow

1. Employee logs into the system
2. Creates an indent request
3. Request is stored in the database
4. Admin reviews the request
5. Admin approves or rejects
6. Employee views status and reports

---

## 🗄️ Database Design

### Tables:

* **emp** → Employee details
* **item** → Indent request details

### Key Fields:

* Status: `Pending`, `Accepted`, `Not Accepted`

---

## 💻 Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Tools: VS Code, phpMyAdmin
